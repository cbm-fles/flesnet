
Three components:
- stsender
- tsbuilder
- tssched

# Draft protocol

stsender <-> tsbuilder
----------------------

-> REGISTER
     hostname, pid
<- OK

-> NEW_SUBTIMESLICE
     serialized descriptor
     rkey
<- OK

<- RELEASE_SUBTIMESLICE
-> OK

-> UNREGISTER
<- OK

(What happens if one of the partners stops unexpectedly?
Should we do heartbeating?)


tsbuilder <-> tssched
---------------------

-> REGISTER
     hostname, pid
<- OK

-> MEMORY_AVAILABLE
     size
<- OK

<- NEW_TIMESLICE
     serialized descriptor
-> OK

-> RELEASE_TIMESLICE / FAILED_TIMESLICE(?)
     timeslice id
     success state
<- OK

-> UNREGISTER
<- OK
